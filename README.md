# Curso Introdutório: Claude Code para Leigos

**Bem-vindo ao seu guia completo de Claude Code!**

Este curso foi criado para pessoas sem experiência em programação ou ferramentas de desenvolvimento. Em módulos curtos e práticos, você vai aprender a usar o Claude Code para automatizar tarefas, criar arquivos, entender código e muito mais — tudo através de uma conversa natural.

---

## Estrutura do Curso

| Módulo | Título | Duração estimada |
|--------|--------|-----------------|
| 0 | Preparação: O que você precisa antes de começar | 15 min |
| 1 | O que é o Claude Code e por que ele é diferente | 20 min |
| 2 | Instalação passo a passo | 30 min |
| 3 | Sua primeira sessão: conversando com o Claude Code | 20 min |
| 4 | Trabalhando com arquivos e pastas | 30 min |
| 5 | Pedindo ao Claude para escrever e editar código | 30 min |
| 6 | Entendendo projetos existentes | 25 min |
| 7 | Automatizando tarefas do dia a dia | 35 min |
| 8 | Dicas, boas práticas e próximos passos | 20 min |

**Tempo total estimado:** ~3,5 horas

---

## Pré-requisitos

- Um computador com Windows, Mac ou Linux
- Conexão com a internet
- Uma conta na Anthropic (criamos juntos no Módulo 2)
- Nenhum conhecimento de programação necessário!
