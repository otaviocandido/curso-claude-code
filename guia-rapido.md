# Guia Rápido — Claude Code para Leigos

> Cole este arquivo em qualquer lugar de fácil acesso para consulta rápida.

---

## Iniciar o Claude Code

```bash
cd sua-pasta-de-trabalho
claude
```

---

## Comandos especiais

| Comando | Ação |
|---------|------|
| `/help` | Lista todos os comandos |
| `/exit` | Sai do Claude Code |
| `/clear` | Limpa a tela |
| `/compact` | Comprime conversa longa |
| `/cost` | Mostra custo da sessão |

---

## Pedidos mais usados

**Criar arquivo:**
```
Crie um arquivo [nome] com [conteúdo]
```

**Ler e resumir:**
```
Leia o arquivo [nome] e me faça um resumo
```

**Organizar pasta:**
```
Organize os arquivos desta pasta por [critério].
Me mostre o plano antes de executar.
```

**Criar script:**
```
Crie um script Python que [descrição do que deve fazer].
Explique o que faz antes de executar.
```

**Corrigir erro:**
```
Esse código deu o seguinte erro: [cole o erro].
Pode consertar?
```

**Entender projeto:**
```
Me dê uma visão geral desta pasta em linguagem simples.
```

---

## Mantra de segurança

1. **Leia** o plano do Claude antes de aprovar
2. **Faça backup** antes de operações em massa
3. **Teste** em dados fictícios antes dos reais
4. **Peça prévia** com "me mostre antes de executar"

---

## Estrutura de pasta recomendada

```
meu-projeto/
├── CLAUDE.md          ← instruções permanentes para o Claude
├── dados/             ← arquivos de entrada
├── resultados/        ← saídas geradas
├── scripts/           ← scripts criados pelo Claude
├── backups/           ← cópias de segurança
└── README.txt         ← anotações suas
```
